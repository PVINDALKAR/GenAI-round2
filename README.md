# GenAI-round2

⚕️ Med-Translate AI: Domain-Specific Medical Report Translator

🚀 Project Overview
Med-Translate AI is a specialized agentic system designed to bridge the communication gap between complex medical diagnostics and patient understanding. It automates the extraction, cleaning, and localization of medical reports into vernacular languagHackathon Category:** Domain-Specialized AI Agents with Compliance Guardrails (Problem Statement #5)[cite: 58, 60].

✨ Key Features
- PDF Intelligence: Extracts text from complex medical layouts using PyPDF2[cite: 63].
- High-Fidelity Translation:Powered by `deep-translator` for stable, context-aware localization[cite: 111].
- Interactive Dashboard: A clean, dual-column Streamlit interface for seamless user experience.
- [cite_Compliance & Safety:Integrated medical disclaimers and data privacy guardrails[cite: 60, 71].

🛠️ Tech Stack
- Core: Python 3.12
- Interface: Streamlit (Web Framework)
- Libraries:PyPDF2 (Extraction), deep-translator (Localization)
- Environment:Windows/Linux/macOS
